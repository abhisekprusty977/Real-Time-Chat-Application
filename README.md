# Real-Time-Chat-Application
It is a modern SwiftUI chat application that delivers secure messaging, Face ID authentication, integrated mini-games, and customizable themes — designed for a fun, private, and engaging iOS experience.

🚀 Features
Secure Login & Face ID:
Protect sensitive content with biometric authentication for enhanced privacy.

Real-Time Chat:
Send and receive messages seamlessly in interactive chat rooms.

Games Menu:
Enjoy built-in mini-games including Tic-Tac-Toe, Memory Match, Rock-Paper-Scissors, and Trivia Quiz.

Customizable Settings:
Personalize app themes, manage Face ID locking, update profiles, and more.

Animated Splash Screen:
Smooth, eye-catching intro animations for a polished, premium feel.

UIKit & SwiftUI Integration:
Core features built using both frameworks for optimal performance and flexibility.
